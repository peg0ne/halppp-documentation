Peekable

Peekable is an object used to make a list like object
it can be used to traverse tokens and such in a compiler

Constructors

Peekable<T> vector<T> value => Peekable<T>
Peekable string value => Peekable
Peekable<T> => Peekable<T>

Functions

index => int
size => int
is_end => bool
is_begin => bool
at int at => Option<T>
prev => Option<T>
prev_peek => Option<T>
next => Option<T>
peek => Option<T>

Examples

let peekable_str = Peekable("Test") //return None

select c from peekable_str.next() do print(c) //will print "T"
select c from peekable_str.next() do print(c) //will print "e"
select c from peekable_str.peek_prev() do print(c) //will print "T"
select c from peekable_str.peek() do print(c) //will print "s"
select c from peekable_str.next() do print(c) //will print "s"
select c from peekable_str.next() do print(c) //will print "T"
//will not print anything since there's no more chars
select c from peekable_str.next() do print(c)
