List

List is class similar to vector in c++ but i like this also

Constructors

List<T> => List<T>
List<T> vector<T> value => List<T>

Functions

at int at => T
at_ptr int at => T*
set int at, T new_value => void
first => Option<T>
last => Option<T>
begin => iterator<T>
end => iterator<T>
len => int
size => int
add T new_value => void
pop => void
clear => void
as_vec_ptr => vector<T>*
insert int at, T new_value => void
remove int at => void

Examples

let list = List<string>({"world", " ", "hello"}) //initialize list

foreach s in list do print(s) //will print "world hello"

list.remove(0) //removes "world"
list.add(0) //adds "world"
list.insert(1) //inserts " " in between "hello" & "world"
foreach s in list do print(s) //will print "hello world"
